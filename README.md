# **База данных сотрудников**

Консольное приложение для создания, хранения и управления информацией о сотрудниках организации.

---

##  **Назначение**

Проект демонстрирует практическое применение структур данных, файлового ввода-вывода, алгоритмов поиска и сортировки в языке C. Может использоваться как учебный пример для изучения работы с динамическими структурами, файлами и алгоритмами обработки данных. Также может служить основой для простой системы управления персоналом.

---

##  **Функциональные возможности**

### **Основной функционал:**

-  **CRUD-операции:** Создание, чтение, обновление записей сотрудников
-  **Поиск:** По фамилии сотрудника
-  **Сортировка:** По фамилии, должности или дате приема
-  **Сохранение:** Бинарный формат для долговременного хранения
-  **Консольный интерфейс:** Интерактивное меню с 10 командами
-  **Расчет стажа:** Определение сотрудников со стажем более 10 лет

### **Структура данных сотрудника:**

| Поле | Тип | Описание |
|------|-----|----------|
| `surname` | `char[50]` | Фамилия сотрудника |
| `name` | `char[50]` | Имя сотрудника |
| `otchestvo` | `char[50]` | Отчество сотрудника |
| `position` | `char[50]` | Должность |
| `gender` | `char[10]` | Пол (муж/жен) |
| `hire_date.day` | `int` | День приема на работу |
| `hire_date.month` | `int` | Месяц приема на работу |
| `hire_date.year` | `int` | Год приема на работу |

##  **Архитектура проекта**

### **Модульная структура:**
```
├── main.c                    # Главный файл с меню и управлением программой
├── database.h               # Объявление структур Employee, Date и функций работы с данными
├── database.c               # Реализация CRUD, поиска и сортировки сотрудников
├── fileio.h                 # Объявление функций для работы с файлами
├── fileio.c                 # Реализация загрузки/сохранения данных в файлы
├── utils.h                  # Объявление вспомогательных функций
├── utils.c                  # Реализация расчета стажа, очистки буфера
├── compile.bat              # Скрипт для компиляции на Windows
├── employees.bin            # Файл базы данных сотрудников
└── README.md                # Инструкция по сборке и использованию
```

### **Ключевые функции:**

1. **Основные функции:**
   - `main()` - точка входа, управление меню
   - `print_employee()` - вывод информации о сотруднике

2. **Операции с данными:**
   - `add_employee()` - добавление нового сотрудника
   - `edit_employee()` - редактирование существующей записи
   - `print_all()` - вывод всех записей
   - `search_employee()` - поиск по фамилии

3. **Сортировка:**
   - `sort_array()` - главная функция сортировки
   - `compare_surname()` - сравнение по фамилии
   - `compare_position()` - сравнение по должности
   - `compare_date()` - сравнение по дате приема

4. **Работа с файлами:**
   - `load_from_file()` - загрузка данных из файла
   - `save_to_file()` - сохранение данных в файл
   - `input_file()` - чтение из файла
   - `output_file()` - запись в файл

##  **Сборка и запуск**

### **Требования:**
- Компилятор C (GCC/Clang/MSVC)
- Стандарт C99 или новее
- Операционная система: Windows/Linux/macOS

### **Инструкция по сборке:**

```bash
# 1. Создайте файл с исходным кодом
# (скопируйте код из репозитория в файл main.c)

# 2. Компиляция
gcc -std=c99 -o employees main.c

# 3. Запуск
./employees        # Linux/macOS
employees.exe      # Windows
```

### **Использование:**
1. Запустите программу
2. Следуйте инструкциям в консольном меню
3. При первом запуске автоматически создадутся тестовые данные
4. Для выхода выберите пункт 10
5. **Не забудьте сохранить данные перед выходом**

##  **Особенности реализации**

### **1. Структура данных**
```c
typedef struct {
    int day, month, year;
} Date;

typedef struct {
    char surname[50];
    char name[50];
    char otchestvo[50];
    char position[50];
    char gender[10];
    Date hire_date;
} Employee;
```

### **2. Алгоритмы поиска**
- Линейный поиск по фамилии в функции `search_employee()`
- Простая и понятная реализация

### **3. Алгоритмы сортировки**
- Использование стандартной функции `qsort()`
- Три варианта сортировки через callback-функции
- Стабильная сортировка по дате (год → месяц → день)

### **4. Работа с файлами**
- Бинарный формат для эффективного хранения
- Проверка существования файлов
- Создание тестовых данных при первом запуске

### **5. Обработка ошибок**
- Проверка границ массива
- Проверка существования файлов
- Защита от переполнения буфера
